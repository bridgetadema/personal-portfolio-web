Artisan Database Project

The Artisan Database Project is a comprehensive web application designed to connect skilled artisans with potential clients.
This platform serves as a web page where artisans can showcase their skills, portfolios, and contact information,
while clients can easily search for and hire artisans based on specific criteria.
The project aims to support local artisans by providing them with an online presence and facilitating easier connections with clients.
